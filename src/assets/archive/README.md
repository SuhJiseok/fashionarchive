Put archive images in this folder.

Use the AI look code parts as folders:

- `st` = style family, for example Street
- `fw` = season, Fall/Winter
- `001` = look sequence

Recommended structure:

```txt
src/assets/archive/st/fw/001/front.png
src/assets/archive/st/fw/001/back.png
src/assets/archive/st/fw/001/side.png
```

Legacy flat filenames still work as a fallback:

```txt
src/assets/archive/st-fw-001-front.png
src/assets/archive/st-fw-001-back.png
src/assets/archive/st-fw-001-side.png
```

Supported formats: jpg, jpeg, png, webp, avif.
