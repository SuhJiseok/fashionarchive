Put product images in this folder.

Use the same archive code folder structure, then use the item `id` as the filename.

Example:

```txt
src/assets/products/st/fw/001/outer.png
src/assets/products/st/fw/001/shoes.webp
```

You can still reference a custom file from `src/data/archiveItems.json` with the
`productImage` field when needed.

Supported formats: jpg, jpeg, png, webp, avif.
